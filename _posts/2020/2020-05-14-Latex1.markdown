# Latex에 지수를 써보자

블로그를 시작한 지 벌써 1년이 다 되어가는데, 여지껏 지수를 $$x^n$$, $$x^2$$과 같이 한 자리 표현만 썼었다. 어제 포스트를 작성하면서 처음으로 $$2^{128}$$처럼 두 자리 이상의 수를 지수로 쓰게 됐다. 그런데 숫자가 똑바로 안 나오고 $$2^128$$ 꼴로 나와서 당황했다. 열심히 구글링을 한 결과

<br>

```latex
2^128
```

<br>

처럼 숫자로 바로 쓰면 안 되고,

<br>

```latex
2^{128}
```

<br>

처럼 숫자를 중괄호로 묶어주면 된다.

참 쉽죠?
